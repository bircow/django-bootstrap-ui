===================
django-bootstrap-ui
===================

This aims to be a powerful Django app to ease the integration of the
popular Bootstrap UI framework (http://getbootstrap.com).

Quick start
-----------

1. Add ``bootstrap_ui`` to your ``INSTALLED_APPS`` setting
   like this::

    INSTALLED_APPS = (
        ...
        'bootstrap_ui',
        ...
    )
